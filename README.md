# webStaging

## 项目基本命令

1. `npm run new` 创建新项目
1. `npm run current` 查看当前项目
1. `npm run switch` 选择开发和打包的版本
1. `npm run delete` 删除某个项目
1. `npm run analyz` 分析当前打包环境，对优化给出建议
1. `npm run dev` 开发环境
1. `npm run build` 生产环境
